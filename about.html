About


This prototype showcases the implementation and integration of RESTful APIs to explore how APIs could be used to help users conveniently retrieve rich-media and associated metadata. Future developments will include full integration of video file and audio files where the user can view or listen without navigating to external sites, as well as the inclusion of more robust metadata.

ChatGPT-5.2 was used to assist in writing the API integration script and to help ensure responsive styling across different screen sizes and devices. It also generated a JavaScript Fetch API template for returning data objects and associated metadata. The model supported initial functionality and layout development for the search form’s response objects. All concepts, written content, and final styling decisions were created by the author.

Poly Haven

Poly Haven is an open 3D Asset repository that is particularly useful to video game designers. One can search for 3D high-definition resolution images, textures, and models of objects they might use in development.

Wiki Commons

This tool uses the MediaWiki Action API to access data from WikiMedia commons.
